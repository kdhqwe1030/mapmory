# Mapmory - Claude 운영 규칙

## 필수 선행 작업
모든 작업 시작 전 아래 파일을 반드시 읽는다.

1. `.claude/AI_WORKFLOW.md` — 작업 규칙, 문서화 방식, 품질 체크
2. `.claude/context/project.md` — 프로젝트 현재 상태 및 개발 단계
3. 작업 유형에 따라 추가로 읽는 파일:
   - UI/컴포넌트/화면 작업 → `.claude/context/frontend.md`
   - 파일 생성/구조 결정 → `.claude/context/folder-structure.md`
   - 진행 중인 작업이 있을 때 → `.claude/tasks/` 폴더 확인

## 프로젝트 요약
- 서비스: 지도 기반 데이트 장소 아카이브
- 스택: Next.js (App Router), TypeScript, Tailwind CSS v4, Naver Map API
- 현재 단계: 2단계 개발 시작 전 (카테고리/방문관리/TODO)
- 상세 내용: `.claude/context/project.md` 참고
